1. First download Xampp and install it.
2. Create a folder name 'Foodforall' under xampp>htdocs 
3. Copy the project files from the repositorie and pest them into the folder named 'Foodforall' under htdocs.
4. Open xampp control panel and click start to Apache and MySQL.
5. Open  browser and type localhost/foodforall to open the project.
6. Download the 'foodforall.sql' file from documents.
7. Open localhost/phpmyadmin and create new DB name foodforall
8. Go inside the DB and import the foodforall.sql file for DB.
